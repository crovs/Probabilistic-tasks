Task 8 Solutions

Current example with repetition allowed
Current all-different example

1. Number of 5-digit PIN codes if digits may repeat

10 * 10 * 10 * 10 * 10 = 10^5 = 100000

A PIN code may start with 0, so even the first position has 10 choices. Since each position is independent and repetition is allowed, the product rule gives 10 choices five times.

3. Number of such codes with all digits different

10 * 9 * 8 * 7 * 6 = 30240

Here order still matters, but repeated digits are forbidden. So after choosing one digit, the next position has one fewer option.

2. Number of such codes containing at least one repeated digit

100000 - 30240 = 69760

The repeated group is easiest to count by complement: start with all PIN codes, then subtract the all-different ones.

Theoretical probabilities: repeated = 69760 / 100000 = 0.6976, all different = 30240 / 100000 = 0.3024

What Must Be Understood

  • A PIN code is an ordered sequence, not a set.
  • Leading zero is allowed, so the first position still has 10 choices.
  • Order matters, so changing positions creates a different code.

Why This Visual Exists

  • It shows why the left product stays 10,10,10,10,10.
  • It shows why the right product shrinks to 10,9,8,7,6.
  • That visual difference is the whole reason the counts are different.

Fast Oral Logic

  • Count all PIN codes first.
  • Count all-different codes second.
  • Use subtraction to get the repeated ones.
