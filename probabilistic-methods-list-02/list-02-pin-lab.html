Why Leading Zero Is Allowed

  • A PIN code is an identifier, not a decimal number.
  • So codes like 04273 are valid outcomes.
  • That is why the first position still has 10 choices.

Why 30240?

  • Start with 10 choices for the first position.
  • Then 9, 8, 7, and 6 because digits cannot repeat.
  • This counts ordered selections without repetition.

Why Use Subtraction?

  • Counting repeated codes directly is awkward.
  • It is easier to count all codes, then subtract the all-different ones.
  • That gives 69760 repeated codes.
