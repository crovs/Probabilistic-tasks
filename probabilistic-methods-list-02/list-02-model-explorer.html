Same Draw Rewritten Three Ways

The extraction above stays the same. These cards show how the recorded outcome changes when we keep labels, keep order, or ignore both.

Labels And Colors

R1R1 means one specific red ball.
B3B3 means one specific blue ball.
G2G2 means one specific green ball.

Animation States

  • bright ball -> still inside the urn
  • faded ball -> already removed
  • highlighted ball -> currently being drawn
  • draw slots -> the physical order of extraction

How To Read The Screen

  • left panel: all 12 physical balls before and after removal
  • draw slots: the actual extraction sequence
  • cards below: the same draw rewritten under each recording rule
  • right panel: the currently highlighted counting model

Same Physical Experiment

  • the urn contents do not change across the three models
  • the draw is always three balls without replacement
  • only the mathematical recording rule changes

Recording Rule

  • sometimes labels such as R2 and B4 matter
  • sometimes only the color pattern matters
  • sometimes extraction order matters and sometimes it is removed

Counting Consequence

  • different outcome meanings create different sample spaces
  • different sample spaces require different counting formulas
  • that is why 220, 27, and 10 all come from the same experiment
