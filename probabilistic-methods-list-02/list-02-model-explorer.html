Probabilistic Methods - List 02 Model Explorer

This page is designed for tutorial presentation. It helps explain how to choose the correct counting model and how to evaluate the main combinatorial formulas used in probability problems.

What To Understand

The goal is not to memorize formulas mechanically. The goal is to identify what the outcome looks like and let that determine the counting model.

What To Visualize

Picture a line of positions, a round table, a selected group, a podium, or a sequence of code boxes. The picture should tell you which model applies.

What To Say

Start with: "We are counting ..., order does/does not matter, repetition is/is not allowed, therefore the correct model is ...".

Decision Assistant

Answer the same questions that appear in the lecture notes: are we using all objects, does order matter, is repetition allowed, are some objects identical, and is the arrangement circular?
